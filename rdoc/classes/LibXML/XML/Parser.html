parse(string, path, attribute)

Call LibXML parse function in fork to avoid memory leak. Method is detailed in: xaop.com/blog/2008/04/01/libxml-ruby-memory-leaks/.

string:the XML to parse.
path:the path to find in the XML.
attribute:the attribute to request per element of the XML.

Returns array of elements from XML parseed by retrieving attribute value for each element in path of XML.
